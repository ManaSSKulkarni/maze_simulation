# Computer Graphics Maze Project

This project was created as part of a Computer Graphics course. It
contains a simple interactive maze implemented using **HTML**, **CSS**,
and **JavaScript**.

## Files

-   **index.html** --- Entry point of the project. Sets up the canvas
    and loads the maze logic.
-   **maze.js** --- Contains the maze generation and rendering logic.

## How to Run

1.  Download or clone this repository.
2.  Open `index.html` in any modern web browser (Chrome, Firefox, Edge).
3.  Interact with the maze as intended.

## Technologies Used

-   HTML5 Canvas
-   JavaScript ES6
-   Basic Computer Graphics Concepts

## Usage

Feel free to modify the maze generation logic or extend the project with
additional graphics features.
